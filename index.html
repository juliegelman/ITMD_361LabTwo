Course Catalog Description

This course will cover the creation of Web pages and sites using HTML, CSS, Javascript, jQuery, and graphical applications as well as the client and server architecture of the Internet and related web technologies. The creation and deployment of modern, standards-compliant web pages are addressed. Students create and deploy a Web site with multiple pages and cross-linked structures. Prerequsites: [none] Credit: 2-2-3 Semester Hours.

Course Outcomes

Students completing this course will be able to:

  • Recognize and differentiate HTML, CSS, and JavaScript mark-up/code
  • Select the proper mark-up tags or code to achieve a particular result
  • Identify improper mark-up and code
  • Produce modern standards-compliant web pages
  • Deploy web pages to a public server
  • Write valid, semantic HTML using an online validator
  • Track progress and development of a project using version control

Course Obectives:

At the conclusion of this course, each successful student will be able to:

  • Explain the client and server architecture of the Internet and related web technologies
  • Use a basic text editor and other software tools to create web pages using HTML, CSS, JavaScript, jQuery, and images
  • Deploy web pages to a GitHub server so they can be viewed live
  • Become comfortable enough that they can argue for/against particular coding methods
  • Write brief descriptive messages to help keep track of commits using version control
